# UnityDOTS SpaceShoot

这是一个Unity Dots相关的游戏Demo，旨在练习如何将一个基于MonoBehavior的游戏，转换为DOTS为基础的Demo。

游戏操作很简单，点击屏幕控制飞船的位置，并开始射击。

尽情享受吧！
